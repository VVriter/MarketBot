## Installation

### Step 1: Clone the Repository
Clone the repository to your local machine:
```bash
git clone https://github.com/VVriter/MarketBot.git
cd MarketBot
```

### Step 2: Enviroment
Copy example .env file and change variables to yours:
```bash
cp .env.sample .env
```

### Step 3: Deploy
Start docker compose and enjoy:
```bash
docker-compose up -d
```
